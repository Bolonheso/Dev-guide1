JavaScript

Um pacote completo e indicado para iniciantes

Por que JavaScript?

O JavaScript é uma linguagem extremamente amigável para quem está começando com desenvolvimento de software. Um simples editor de texto como o Visual Studio Code já supre muito bem as necessidades

Possui muitos frameworks e bibliotecas sendo muito utilizado no mercado de trabalho o que se traduz em um ecossistema de desenvolvimento muito rico e dinâmico.

A quantidade de pacotes, bibliotecas e frameworks open source para JavaScript existentes já mostra o tamanho da comunidade JavaScript e o quão ativa ela é: trata-se de uma das comunidades de desenvolvimento mais ativas.

O JavaScript é uma das linguagens mais versáteis na atualidade, o que possibilita a sua utilização em uma série de ambientes diferentes. Isso se reflete na quantidade de vagas disponíveis para desenvolvedores

O que o JavaScript faz?

O JavaScript serve para implementar itens complexos nos sites, mais do que apenas elementos estáticos

Além do JavaScript puro, são usados frameworks para aumentar o desempenho, como o ReactJs
